### Intro

- 기존의 테슬라 홈페이지를 토대로 된 홈페이지 구현 계획

---

#### Main Func

1. 회원관리

   - 회원가입, 로그인, 로그아웃

2. 상품 장바구니 관리 기능

   - 각 상품 별로 장바구니에 추가 기능
   - 개개인의 장바구니 내역
   - 상품 삭제 기능

3. 상품 검색 기능
   - 사용자가 원하는 상품을 자유롭게 검색
   - 입력한 검색어를 포함한 상품들 전부 나열
